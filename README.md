This repository contains small React/Next.js projects created while learning React. Each project is in its own folder with its own README.

## Projects Included

### 1. ToDo_App
A To-Do List application built with React and Next.js.  
**Features:**
- Add, delete, and manage daily tasks
- State management with React Hooks
- Styled with CSS and Tailwind (if applicable)

### 2. password_generator_buddy
A password generator app built with React.  
**Features:**
- Generate passwords of variable length
- Option to include numbers and special characters
- Copy to clipboard functionality
- Built with React Hooks and Tailwind CSS
